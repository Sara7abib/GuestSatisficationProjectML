# Guest Satisfaction Prediction

## Overview
This project explores the key factors that influence guest satisfaction in Airbnb listings. Using machine learning techniques, we analyze listing characteristics, host information, pricing, amenities, and review-related features to predict guest satisfaction scores and identify the most impactful factors.

## Objectives
- Analyze Airbnb listing data.
- Perform data preprocessing and feature engineering.
- Build and compare regression and classification models.
- Evaluate model performance.
- Identify the features that most influence guest satisfaction.

## Dataset
The dataset contains information about Airbnb listings, including:
- Property details
- Host information
- Pricing
- Amenities
- Review scores
- Location
- Listing descriptions

## Data Preprocessing
- Handling missing values
- Removing duplicates
- Encoding categorical features
- Feature scaling
- Feature engineering
- Outlier handling

## Machine Learning Models

### Regression
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor

### Classification
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- Gradient Boosting Classifier

## Evaluation Metrics

### Regression
- Mean Squared Error (MSE)
- R² Score

### Classification
- Accuracy
- Precision
- Recall
- F1 Score

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Results
The models were compared based on their evaluation metrics to determine the best-performing approach. Feature importance analysis was conducted to identify the main drivers of guest satisfaction.
